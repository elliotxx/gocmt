## gocmt

你对 go 代码质量有更高的要求吗？那么注释一定是必不可少的。

`gocmt` 支持检查、补充、优化 go 代码注释，同时支持计算代码注释率，用于衡量项目是否有充足的注释说明。

## Usage

```
gocmt -f example.go
gocmt -f path/to/foo/
```
